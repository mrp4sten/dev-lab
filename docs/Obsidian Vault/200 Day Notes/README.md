# 200 Day Notes

## Purpose

This is your **daily journal** for tracking study sessions, reflections, progress, and habits. Each day gets its own note.

## How to Use

1. **Create a new note each day** using the `005 Daily Note` template from `700 Templates/`.
2. **Set your top 3 goals** at the start of each day.
3. **Log your study sessions** including reading, coding practice, problem solving, and project work.
4. **Reflect at the end of the day** on what went well, what was challenging, and what to carry forward.
5. **Track your habits** using the checklist section (reading, coding, problem solving, note review, etc.).

## What Belongs Here

- Daily study logs and reflections
- Weekly review notes (use the `004 Weekly Review` template)
- Maker journal entries (use the `008 Make journal` template)

## Naming Convention

Name files by date for easy chronological sorting, e.g.:
- `2026-02-17 Monday`
- `Week 08 Review`

## Tips

- Be honest about energy levels, focus quality, and motivation -- this helps you identify your best working patterns.
- Use the "Carry Forward" section so unfinished tasks don't get lost.
- Link to concepts studied and problems solved to build connections across the vault.
- Review previous days to spot trends in productivity and learning.
