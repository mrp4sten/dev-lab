# Cheatsheets

This directory contains quick reference guides and command cheatsheets for various tools and technologies.

## What Goes Here

- **Command References**: Quick syntax guides for CLI tools
- **API References**: Common API patterns and examples
- **Language Syntax**: Programming language quick references
- **Framework Guides**: Key concepts and patterns for frameworks
- **Troubleshooting**: Common issues and solutions

## Naming Convention

Use descriptive filenames with lowercase and hyphens:
- `git-commands.md` - Git command reference
- `docker-cheatsheet.md` - Docker commands and patterns
- `javascript-syntax.md` - JavaScript syntax guide
- `api-testing.md` - API testing patterns

## Format Guidelines

Each cheatsheet should include:
1. **Purpose** - What this cheatsheet covers
2. **Common Commands/Patterns** - Most frequently used items
3. **Examples** - Real-world usage examples
4. **References** - Links to official documentation

## Contributing

When adding new cheatsheets:
- Keep them concise and actionable
- Focus on practical, frequently-used information
- Include copy-paste ready examples
- Cross-reference with related cheatsheets
